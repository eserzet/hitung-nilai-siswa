# 📊 nuba grade || Sistem Perhitungan Nilai Siswa SMA Informatika Nurul Bayan

Selamat datang di **nuba grade**, sebuah aplikasi web yang dirancang khusus untuk membantu guru dan siswa SMA Informatika Nurul Bayan dalam menghitung nilai secara otomatis dan efisien. Aplikasi ini juga dilengkapi dengan fitur cetak dan unduh hasil dalam format PDF.

🔗 Live demo: [kalkulator-nilai-siswa.vercel.app](https://kalkulator-nilai-siswa.vercel.app/)

---

## ✨ Fitur Unggulan
- **Input Data Siswa**: Memungkinkan input nama siswa dan pemilihan kelas (X, XI, XII).
- **Perhitungan Nilai Otomatis**: Menghitung rata-rata dari 15 mata pelajaran berikut:
  - Matematika
  - Bahasa Indonesia
  - Bahasa Inggris
  - Fisika
  - Biologi
  - Kimia
  - Geografi
  - Sejarah
  - Ekonomi
  - Sosiologi
  - Pendidikan Agama
  - Pancasila
  - Seni Budaya
  - Informatika/Algoritma
  - Prakarya
- **Validasi Nilai Real-time**: Memastikan nilai yang dimasukkan berada dalam rentang 0-100.
- **Analisis Hasil Komprehensif**: Menyajikan total nilai, rata-rata, predikat (A/B/C/D), dan status kelulusan (Lulus/Tidak Lulus).
- **Antarmuka Pengguna Intuitif**: Desain UI yang menarik dengan animasi, tema gradien, dan responsif di berbagai perangkat (desktop dan mobile).
- **Fitur Cetak dan Unduh PDF**: Memungkinkan pengguna untuk mencetak hasil perhitungan langsung atau mengunduhnya dalam format PDF menggunakan library jsPDF.

---

## 🛠️ Teknologi yang Digunakan
- **HTML5**: Struktur dasar halaman web.
- **CSS3**: Styling dan tata letak responsif (menggunakan Flexbox dan animasi).
- **JavaScript (Vanilla)**: Logika perhitungan dan interaktivitas aplikasi.
- **jsPDF Library**: Untuk fungsionalitas pembuatan dan pengunduhan dokumen PDF.
- **Deployment**: Vercel (untuk live demo).

---

## ✅ Cara Menggunakan
1.  Akses aplikasi melalui link live demo atau kloning repositori ini dan buka `index.html` di browser Anda.
2.  Masukkan **Nama Siswa/Siswi** dan pilih **Kelas** yang sesuai.
3.  Isi nilai untuk setiap mata pelajaran (pastikan nilai antara 0-100).
4.  Tekan tombol **Hitung** untuk melihat hasil perhitungan, termasuk rata-rata, total, predikat, dan status kelulusan.
5.  Gunakan tombol **Cetak** untuk mencetak hasil langsung atau **Download PDF** untuk menyimpan laporan dalam format PDF.
6.  Untuk memulai perhitungan baru, tekan tombol **Reset**.
